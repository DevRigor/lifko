# Context AI

## Proposito
Esta carpeta concentra contexto operativo del proyecto para asistentes, agentes y desarrolladores que necesiten entender rapidamente la aplicacion.

## Estructura
- `architecture/`: contexto tecnico general y decisiones de implementacion
- `modules/`: contexto por modulo o seccion visible del sitio
- `business/`: reglas y objetivos funcionales
- `design/`: guia visual, tema y comportamiento UX
- `integration/`: integraciones externas y contratos practicos

## Uso recomendado
- revisar `architecture/project-structure.md` para una vista general tecnica
- revisar `modules/*.md` para trabajar una seccion especifica
- revisar `business/*.md` antes de cambiar servicios, mensajes o flujos de captura
- revisar `design/*.md` antes de tocar tema, componentes o navegacion
